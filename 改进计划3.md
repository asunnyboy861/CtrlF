# CtrlF 改进计划 3

> **日期**: 2026-05-15
> **目标**: 新增"从相册导入照片搜索"增值功能，响应用户反馈 "I want to scan from camera live not from photos"
> **原则**: 单一职责、高内聚低耦合、代码复用、最小代码、风格一致、优先继承

---

## 需求背景

### 用户反馈
美国客户反馈："I want to scan from camera live not from photos"

### 反馈解读
经深度分析，存在两种可能理解：

| 理解 | 可能性 | 说明 |
|------|--------|------|
| **理解A**: 用户误解应用工作方式 | 🟡 中 | 用户可能认为需要"先拍照再搜索"，而不知道应用已经是实时扫描 |
| **理解B**: 用户希望增加"从相册导入照片搜索"功能 | 🟢 高 | 用户希望除了实时相机扫描外，还能从相册中选取照片进行文本搜索 |

### 需求确认
**采用理解B**：新增"从相册导入照片搜索"功能，原因如下：
1. 用户场景痛点：用户可能已经拍了书本的照片，想直接搜索而不是重新对着书拍
2. 竞品对比：Google Lens、Prizmo Go 都支持从相册导入
3. 用户预期：很多扫描类App都支持"拍照/相册"双入口
4. 技术可行性：极高，核心OCR引擎可完全复用

---

## 技术可行性分析

### 现有架构优势

当前架构的 `TextRecognitionEngine` 和 `HighlightOverlayView` 是完全解耦的：

```
┌─────────────────────────────────────────────────────────────┐
│                    数据流向（单向）                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   输入源 ──► TextRecognitionEngine ──► matchedItems ──► HighlightOverlayView
│     │              │                        │                      │
│     │              │                        │                      │
│     ▼              ▼                        ▼                      ▼
│  Camera        processFrame()         [MatchedTextItem]      Canvas渲染
│  (实时帧)      VNImageRequestHandler                          (高亮框)
│                                                             │
│   ─────────────────────────────────────────────────────────  │
│                                                             │
│  Photo         processImage()          [MatchedTextItem]      Canvas渲染
│  (静态图片)    VNImageRequestHandler                          (高亮框)
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键发现**：
- `TextRecognitionEngine.processFrame()` 接受 `CVImageBuffer`
- `VNImageRequestHandler` 同时支持 `CVImageBuffer` 和 `CGImage`/`UIImage`
- `HighlightOverlayView` 基于 `matchedItems` 渲染，与数据来源无关
- **结论**：只需新增一个接受 `UIImage` 的入口方法，其他逻辑100%复用

### 复用矩阵

| 现有模块 | 复用程度 | 说明 |
|----------|----------|------|
| `TextRecognitionEngine` | 🟢 95% | 新增 `processImage()` 方法，复用 `handleRecognitionResults()` |
| `HighlightOverlayView` | 🟢 100% | 完全复用，无需修改 |
| `SearchViewModel` | 🟢 90% | 新增搜索模式状态，复用搜索逻辑 |
| `SearchBarView` | 🟢 100% | 完全复用，无需修改 |
| `StoreManager` | 🟢 100% | 复用 `ProFeature` 枚举，新增 `.photoImport` |

### 模块 2: TextRecognitionEngine 扩展 (静态图片OCR)

#### 需求描述
扩展现有的 `TextRecognitionEngine`，新增处理静态图片的方法，复用现有的OCR识别和匹配逻辑。

#### 设计原则
- 单一职责：保持原有的文本识别职责
- 代码复用：复用 `handleRecognitionResults()` 方法
- 最小改动：新增方法，不修改现有方法签名

#### 接口设计
```swift
import Vision
import UIKit

extension TextRecognitionEngine {
    
    /// 处理静态图片的文本识别
    /// - Parameter image: 要识别的 UIImage
    /// 复用现有的 handleRecognitionResults() 方法处理识别结果
    func processImage(_ image: UIImage)
}
```

#### 实现规则
1. 将 `UIImage` 转换为 `CGImage`
2. 使用 `VNImageRequestHandler(cgImage:options:)` 创建请求处理器
3. 复用现有的 `VNRecognizeTextRequest` 配置（语言、识别级别等）
4. 调用现有的 `handleRecognitionResults()` 处理结果
5. 保持与 `processFrame()` 相同的搜索查询逻辑

#### 代码示例
```swift
import Vision
import UIKit

extension TextRecognitionEngine {
    
    /// 处理静态图片的文本识别
    /// - Parameter image: 要识别的 UIImage
    /// 说明：与 processFrame() 共享 handleRecognitionResults() 逻辑
    func processImage(_ image: UIImage) {
        guard let cgImage = image.cgImage else { return }
        guard !searchQuery.isEmpty else {
            DispatchQueue.main.async { self.matchedItems = [] }
            return
        }
        
        let request = VNRecognizeTextRequest { [weak self] request, error in
            guard let observations = request.results as? [VNRecognizedTextObservation],
                  error == nil else { return }
            self?.handleRecognitionResults(observations)
        }
        
        // 复用现有配置
        request.recognitionLevel = .accurate
        request.recognitionLanguages = languageMap[ocrLanguage] ?? ["en-US", "en-GB"]
        request.usesLanguageCorrection = true
        request.minimumTextHeight = 0.02
        
        let handler = VNImageRequestHandler(cgImage: cgImage, options: [:])
        DispatchQueue.global(qos: .userInteractive).async {
            try? handler.perform([request])
        }
    }
}
```

### 模块 3: PhotoSearchViewModel (照片搜索视图模型)

#### 需求描述
管理照片搜索模式的视图状态，包括选中的照片、搜索模式切换、相机会话管理等。

#### 设计原则
- 单一职责：只负责照片搜索模式的状态管理
- 继承现有：复用 `SearchViewModel` 的搜索逻辑
- 最小改动：通过组合而非继承扩展功能

#### 接口设计
```swift
import SwiftUI
import PhotosUI

/// 照片搜索视图模型
/// 职责：管理照片搜索模式的状态和行为
@MainActor
class PhotoSearchViewModel: ObservableObject {
    
    // MARK: - Published Properties
    
    /// 当前搜索模式
    @Published var searchMode: SearchMode = .camera
    
    /// 选中的照片（照片模式）
    @Published var selectedPhoto: UIImage?
    
    /// 照片选择器是否显示
    @Published var showPhotoPicker: Bool = false
    
    // MARK: - Dependencies
    
    /// 复用现有的搜索视图模型
    let searchViewModel: SearchViewModel
    
    // MARK: - Initialization
    
    init(searchViewModel: SearchViewModel) {
        self.searchViewModel = searchViewModel
    }
    
    // MARK: - Public Methods
    
    /// 切换到照片模式并打开照片选择器
    func switchToPhotoMode()
    
    /// 切换回相机模式
    func switchToCameraMode()
    
    /// 处理选中的照片
    /// - Parameter image: 用户从相册选择的照片
    func handleSelectedPhoto(_ image: UIImage)
    
    /// 清除当前照片
    func clearPhoto()
}
```

#### 实现规则
1. 切换到照片模式时，暂停相机会话（`cameraManager.stopSession()`）
2. 切换回相机模式时，恢复相机会话（`cameraManager.startSession()`）
3. 选择照片后，调用 `textEngine.processImage()` 进行OCR
4. 照片模式下，搜索文本变化时重新处理同一张照片
5. 清除照片时，清空匹配结果并返回相机模式

#### 代码示例
```swift
import SwiftUI
import PhotosUI

@MainActor
class PhotoSearchViewModel: ObservableObject {
    
    @Published var searchMode: SearchMode = .camera
    @Published var selectedPhoto: UIImage?
    @Published var showPhotoPicker: Bool = false
    
    let searchViewModel: SearchViewModel
    
    init(searchViewModel: SearchViewModel) {
        self.searchViewModel = searchViewModel
    }
    
    /// 切换到照片模式并打开照片选择器
    func switchToPhotoMode() {
        searchMode = .photo
        showPhotoPicker = true
        // 暂停相机以节省资源
        searchViewModel.cameraManager.stopSession()
    }
    
    /// 切换回相机模式
    func switchToCameraMode() {
        searchMode = .camera
        selectedPhoto = nil
        showPhotoPicker = false
        // 恢复相机
        searchViewModel.cameraManager.startSession()
        // 清空匹配结果
        searchViewModel.textEngine.matchedItems = []
    }
    
    /// 处理选中的照片
    func handleSelectedPhoto(_ image: UIImage) {
        selectedPhoto = image
        showPhotoPicker = false
        // 执行OCR识别
        searchViewModel.textEngine.processImage(image)
    }
    
    /// 清除当前照片
    func clearPhoto() {
        selectedPhoto = nil
        searchViewModel.textEngine.matchedItems = []
    }
}
```

#### 测试用例
```swift

import SwiftUI

struct PhotoSearchView: View {
    @ObservedObject var viewModel: PhotoSearchViewModel
    @ObservedObject var storeManager: StoreManager
    @ObservedObject var audioManager: AudioFeedbackManager
    
    var body: some View {
        ZStack {
            Color.black.ignoresSafeArea()
            
            if let photo = viewModel.selectedPhoto {
                // 照片显示层
                photoDisplayView(photo)
                
                // 高亮叠加层（复用现有组件）
                HighlightOverlayView(matchedItems: viewModel.searchViewModel.textEngine.matchedItems)
                    .ignoresSafeArea()
            } else {
                // 空状态
                emptyStateView
            }
            
            // 搜索栏 + 工具栏
            VStack {
                SearchBarView(
                    searchText: $viewModel.searchViewModel.searchText,
                    matchCount: viewModel.searchViewModel.matchCount,
                    isFlashOn: viewModel.searchViewModel.isFlashOn,
                    onFlashToggle: { viewModel.searchViewModel.toggleFlashlight() },
                    remainingFree: viewModel.searchViewModel.remainingFreeSearches,
                    isProUser: viewModel.searchViewModel.isProUser,
                    onSubmit: { }
                )
                .padding(.top, 4)
                
                Spacer()
                
                // 照片模式专用工具栏
                photoModeToolbar
            }
        }
        .onChange(of: viewModel.searchViewModel.searchText) { _, newValue in
            // 照片模式下，搜索文本变化时重新处理照片
            if viewModel.searchMode == .photo, let photo = viewModel.selectedPhoto {
                viewModel.searchViewModel.updateSearchQuery()
                viewModel.searchViewModel.textEngine.processImage(photo)
            }
        }
    }
    
    // MARK: - Private Views
    
    @ViewBuilder
    private func photoDisplayView(_ photo: UIImage) -> some View {
        GeometryReader { geometry in
            Image(uiImage: photo)
                .resizable()
                .scaledToFit()
                .frame(width: geometry.size.width, height: geometry.size.height)
                .position(x: geometry.size.width / 2, y: geometry.size.height / 2)
        }
        .ignoresSafeArea()
    }
    
    private var emptyStateView: some View {
        VStack(spacing: 16) {
            Image(systemName: "photo.on.rectangle.angled")
                .font(.system(size: 64))
                .foregroundStyle(.secondary)
            Text("Select a photo to search")
                .font(.title3)
                .foregroundStyle(.secondary)
            Button("Choose Photo") {
                viewModel.showPhotoPicker = true
            }
            .buttonStyle(.borderedProminent)
            .tint(.yellow)
        }
    }
    
    private var photoModeToolbar: some View {
        HStack(spacing: 24) {
            // 返回相机按钮
            Button {
                viewModel.switchToCameraMode()
            } label: {
                VStack(spacing: 4) {
                    Image(systemName: "camera.fill")
                    Text("Camera").font(.caption2)
                }
                .foregroundStyle(.white)
            }
            
            // 更换照片按钮
            Button {
                viewModel.showPhotoPicker = true
            } label: {
                VStack(spacing: 4) {
                    Image(systemName: "arrow.left.arrow.right")
                    Text("Change").font(.caption2)
                }
                .foregroundStyle(.white)
            }
            
            // 历史记录按钮
            Button {
                viewModel.searchViewModel.showHistory = true
            } label: {
                VStack(spacing: 4) {
                    Image(systemName: "clock.arrow.circlepath")
                    Text("History").font(.caption2)
                }
                .foregroundStyle(.white)
            }
            
            // 设置按钮
            Button {
                viewModel.searchViewModel.showSettings = true
            } label: {
                VStack(spacing: 4) {
                    Image(systemName: "gearshape")
                    Text("Settings").font(.caption2)
                }
                .foregroundStyle(.white)
            }
        }
        .padding(.vertical, 12)
        .padding(.horizontal, 32)
        .background(.ultraThinMaterial, in: RoundedRectangle(cornerRadius: 16))
        .padding(.bottom, 24)
    }
}
```

```swift
    @StateObject private var photoSearchVM: PhotoSearchViewModel  // 🆕 新增
    
    // ... 其他属性 ...

            if viewModel.cameraManager.authorizationStatus == .authorized {

                switch photoSearchVM.searchMode {

                case .photo:

                        viewModel: photoSearchVM,
                        storeManager: storeManager,
                        audioManager: audioManager
                    )
                }

            .ignoresSafeArea()

            .ignoresSafeArea()

            SearchBarView(
                searchText: $viewModel.searchText,
                matchCount: viewModel.matchCount,
                isFlashOn: viewModel.isFlashOn,
                onFlashToggle: { viewModel.toggleFlashlight() },
                remainingFree: viewModel.remainingFreeSearches,
                isProUser: viewModel.isProUser,

            )

                freeSearchLimitBanner
            }

                onCapture: { captureScreenshot() },

                onSettings: { viewModel.showSettings = true },
                onPhotoImport: { handlePhotoImport() },  // 🆕 新增
                canImportPhoto: storeManager.canAccess(.photoImport)  // 🆕 新增
            )
        }
    }

    private func handlePhotoImport() {
        guard storeManager.canAccess(.photoImport) else {
            viewModel.showPaywall = true
            return
        }
        photoSearchVM.switchToPhotoMode()
    }
```
